# CommunityAI

CommunityAI is an AI moderator for community platforms like Discord, Slack, and Microsoft Teams. It analyzes content in real-time and classifies it as appropriate or inappropriate based on predefined and custom rules.

## Project Structure

- `backend/`: FastAPI application
- `frontend/`: Streamlit application
- `integrations/`: Modules for platform integrations
- `ai_model/`: AI model training and prediction scripts
